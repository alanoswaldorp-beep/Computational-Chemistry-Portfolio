# DFT Calculations

## Overview

This section contains a basic Density Functional Theory (DFT) analysis performed on the most promising compound identified during the virtual screening campaign.

The objective of this calculation was not to perform an exhaustive quantum chemical study, but rather to demonstrate the integration of quantum chemistry methods within a computational drug discovery workflow.

---

## Workflow

SMILES Structure
→ 3D Structure Generation
→ Energy Minimization
→ ORCA Input Preparation
→ Geometry Optimization
→ Molecular Orbital Analysis

---

## Methodology

Software:

* ORCA
* Python
* RDKit
* Open Babel
* IboView

Calculation Type:

* Geometry Optimization
* HOMO/LUMO Analysis

The calculation was intentionally kept simple and computationally inexpensive, focusing on the electronic characterization of the selected lead compound.
