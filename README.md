# Computational Chemistry Portfolio 🧪💻

A collection of computational chemistry and cheminformatics projects focused on molecular modeling, virtual screening, ligand design, and quantum chemical analysis.

## Computational Workflow

This portfolio follows a simplified computer-aided drug design (CADD) workflow:

## Project Overview

### 🔹 Validation
Quality assessment and validation steps for computational workflows.

### 🔹 Lead Design
Design and modification of molecular structures for optimization.

### 🔹 Library Generation
Generation and preparation of molecular libraries for screening.

### 🔹 Virtual Screening
Computational evaluation of compound libraries to identify promising candidates.

### 🔹 Hit Selection
Selection and prioritization of potential hits based on computational results.

### 🔹 DFT Analysis
Quantum chemical analysis of selected molecules using Density Functional Theory calculations.
